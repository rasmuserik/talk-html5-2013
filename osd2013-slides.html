Canvas Example

Trace 40 random walks. 
var canvas = document.getElementById("canvas");
var ctx = canvas.getContext("2d");
ctx.fillRect(0, 0, canvas.width, canvas.height);

var buf = ctx.getImageData(0, 0, canvas.width, canvas.height);
for(var i = 0; i < 40; ++i) {
    var x = canvas.width / 2, y = canvas.height / 2;
    for(var j = 0; j < 50000; ++j) {
        x = (x + Math.random() * 5 - 2)|0;
        y = (y + Math.random() * 5 - 2)|0;
        buf.data[x * 4 + y * 4 * canvas.width] += 10; 
    } 
}
ctx.putImageData(buf, 0, 0);

Emscripten • LLVM • C/C++

  • Low-level virtual machine: C/C++
    Fortran, Ada, Haskell, ...
  • Python, Lua, LaTeX, gnuplot, OpenTTD, Cube 2: Sauerbraten, OpenDUNE, zlib, Poppler, FreeType, libSDL, SQLite, Qt, ...
  • Useful for porting existing C/C++ code to run in JavaScript
  • Moderate performance cost
## Getting started 1. Install llvm/clang-3.2 from http://llvm.org/releases/download.html#3.2 2. git clone git@github.com:kripken/emscripten.git 3. emscripten/emcc src.cc -o dst.[html|js]

## Getting started - eclipse-plugin - should be easy for eclipse users, - or: 1. get/extract http://google-web-toolkit.googlecode.com/files/gwt-2.5.0.zip 2. ./gwt-2.5.0/webAppCreator -out AppName com.rasmuserik.AppName 3. cd AppName; ant devmode

Sample canvas program

package com.rasmuserik.client;

import com.google.gwt.core.client.*;
import com.google.gwt.dom.client.*;
import com.google.gwt.canvas.dom.client.Context2d;

public class AppName implements EntryPoint {
  public void onModuleLoad() {
    CanvasElement canvas = (CanvasElement)
        (Element)DOM.getElementById("canvas");
    Context2d ctx = canvas.getContext2d();
    ctx.fillText("Hello HTML5 GWT", 0, 20);
  }
}
